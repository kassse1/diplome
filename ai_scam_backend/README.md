# 🛡️ AI Scam Detector

AI Scam Detector — это веб-приложение для анализа текстовых сообщений и определения уровня риска мошенничества.  
Проект использует гибридный AI-подход, объединяющий семантическое машинное обучение и детерминированные правила для критических сценариев.

---

## ✨ Возможности

- 🌍 Поддержка нескольких языков  
  English · Русский · Español · Português · Қазақша

- 🧠 Семантический AI-анализ на основе sentence embeddings

- ⚠️ Трёхуровневая классификация сообщений:
  - 🟢 **Safe** — безопасное сообщение
  - 🟡 **Dangerous** — потенциально опасное сообщение
  - 🔴 **Scam** — мошенничество

- 📊 Визуализация риска (цвет + confidence)
- 🔗 REST API для интеграции с другими системами

---

🧠 AI-подход

В проекте применяется гибридная модель анализа сообщений:

Rule-based анализ (Safety Net)

Детерминированные правила используются для критических сценариев, где ложные отрицания недопустимы:

банковский фишинг

запросы подтверждения личных данных

имитация уведомлений служб безопасности

Такие сообщения немедленно классифицируются как мошенничество.

Semantic AI (основной AI-компонент)

Для оценки риска используется семантическое сравнение текста с эталонными мошенническими сценариями:

применяется многоязычная модель Sentence Transformers

вычисляется cosine similarity между эмбеддингами

подход устойчив к языку и формулировкам сообщений

Результат интерпретируется как уровень риска в диапазоне от 0.0 до 1.0.
